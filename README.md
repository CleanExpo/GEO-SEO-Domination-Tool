## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- SQLite (development) or PostgreSQL (production)

### Installation

```bash
# Clone repository
git clone https://github.com/unite-group/geo-seo-domination-tool.git
cd GEO_SEO_Domination-Tool

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Generate encryption key for credentials
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
# Add output to .env.local as ENCRYPTION_KEY

# Initialize database
npm run db:init

# Start development server
npm run dev
```

### Environment Variables

```env
# Database
DATABASE_URL=file:./data/geo-seo.db  # SQLite (dev)

# Security
ENCRYPTION_KEY=your-64-char-hex-key-here

# AI Services
OPENAI_API_KEY=sk-proj-...
DEEPSEEK_API_KEY=sk-...
DEEPSEEK_BASE_URL=https://api.openai.com/v1
DEEPSEEK_MODEL=gpt-4o-mini

# Email
EMAIL_PROVIDER=resend
EMAIL_API_KEY=re_...
EMAIL_FROM=noreply@yourdomain.com
```

### First Run

```bash
# Start development server
npm run dev

# Open browser
http://localhost:3000

# Create your first portfolio
http://localhost:3000/onboarding
```
