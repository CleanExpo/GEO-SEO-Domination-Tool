## üìã Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure API keys:**
   ```bash
   cp .env.example .env
   ```

   Add your API keys to `.env`:
   ```
   SEMRUSH_API_KEY=your_semrush_key
   ANTHROPIC_API_KEY=your_anthropic_key
   GOOGLE_API_KEY=your_google_key
   ```

3. **Run development:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build:win
   ```
